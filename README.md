# defaultinator-api

## Getting Started

### Configuration

The following environment variables can be set in a `.env` file or on the command line. An example with all default values is listed below.

```
PORT=3000
```

### Development

```
cd api
npm install
npm run dev
```

### Testing

```
npm test
```

For coverage run:

```
npm run coverage
```
